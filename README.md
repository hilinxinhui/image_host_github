# Image Host GitHub

使用[PicGo]()搭建图床。

## 搭建图床

- 注册GitHub账号
- 新建图床仓库（设置为`Public`）
- 生成token(用`Tokens(classic)`，勾选`repo`权限即可)
- 安装PicGo客户端，填入各种信息
- 上传图片
- [使用图床](#图床测试)

## 图床测试

如图是PicGo的GitHub设置示意图，填写：

- 图床仓库名
- 上传到图床仓库的哪一个分支
- token
- 存储路径和自定义域名（非必须，我设置了存储路径）
